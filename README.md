## вңЁ Overview

**Zenith Aurora Edition** is a showcase website inspired by [Shopify Editions](https://www.shopify.com/editions/winter2026) вҖ” rebuilt from scratch as a **React + Vite** application that integrates **all 11 major React animation libraries** into a single cohesive, scroll-driven experience.

> _Every scroll, every hover, every transition вҖ” powered by a different animation engine working in harmony._

<br/>

<table>
<tr>
<td width="50%">

### рҹҺҜ What It Does

- **Lottie-powered** loading experience
- **3D particle hero** with Three.js
- **Parallax depth layers** on scroll
- **Physics-based** counter animations
- **GSAP scroll-triggered** section reveals
- **Drag-and-drop** canvas demo
- **Animated data charts** with React Move
- **3D card tilt** on mouse tracking
- Fully responsive, dark-themed design

</td>
<td width="50%">

### рҹ§  Why It Exists

This project demonstrates that **11 animation libraries** can coexist in a single React app without conflict вҖ” each handling what it does best:

| Concern | Library |
|---------|---------|
| Layout transitions | Framer Motion |
| Physics springs | React Spring |
| 3D rendering | React Three Fiber |
| Scroll timelines | GSAP |
| Data animation | React Move |
| Reveal effects | React Awesome Reveal |
| Parallax depth | react-scroll-parallax |

</td>
</tr>
</table>

## рҹҡҖ Getting Started

### Prerequisites

- **Node.js** вүҘ 18
- **npm** вүҘ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/zenith-aurora-react.git

# Navigate to the project
cd zenith-aurora-react

# Install dependencies (--legacy-peer-deps required for React 19 compat)
npm install --legacy-peer-deps

# Start the dev server
npm run dev
```

The app will open at **`http://localhost:3000`**.

### Build for Production

```bash
npm run build
npm run preview
```
